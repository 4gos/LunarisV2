Como funciona nuestro predictor

Para analizar imágenes de lesiones en la piel usamos un modelo de inteligencia artificial llamado Vision Transformer, o simplemente ViT. Es una tecnología relativamente nueva en visión por computadora que, en vez de mirar la imagen como un todo desde el principio, la divide en pequeñas partes para poder interpretarla mejor.

Primero, cada imagen dermatológica se separa en pequeños cuadrados (parches). Es como si cortaras una foto en muchas piezas y el modelo analizara cada una por separado. Después, esas piezas se convierten en datos que la red puede entender y se les agrega información sobre su posición para no perder la estructura original de la imagen. Esto es importante porque la ubicación de una lesión o su forma dentro de la piel aporta información clínica relevante.

Una vez preparados esos parches, el modelo usa un mecanismo llamado “autoatención”. Dicho simple: cada parte de la imagen puede “mirar” a las demás para entender cómo se relacionan. Esto permite detectar patrones complejos, por ejemplo combinaciones de color, textura o bordes que pueden indicar distintos tipos de lesiones cutáneas.

Para entrenar el modelo usamos imágenes previamente clasificadas en cinco categorías dermatológicas. Parte de las imágenes se usan para que el modelo aprenda (entrenamiento), otra parte para verificar que realmente esté aprendiendo y no memorizando (validación), y un último grupo para evaluar su desempeño final (prueba). Esta separación ayuda a que los resultados sean confiables.

Además, utilizamos un modelo previamente entrenado con millones de imágenes generales. Esto se conoce como transferencia de aprendizaje. En lugar de empezar desde cero, el modelo ya sabe reconocer formas, colores y texturas básicas, y solo necesita adaptarse al problema específico de lesiones cutáneas. Esto reduce tiempos de entrenamiento y mejora la precisión.

También aplicamos aumentación de datos. Básicamente, generamos variaciones de las imágenes (cambios leves de brillo, rotaciones, recortes, etc.) para que el modelo vea más diversidad. Esto ayuda a evitar que aprenda solo casos muy específicos y mejora su capacidad para generalizar a imágenes nuevas. En clases donde había menos ejemplos se aplicaron transformaciones más intensas para equilibrar el aprendizaje.

Durante el entrenamiento se ajustaron varios parámetros, como la velocidad de aprendizaje del modelo o la regularización para evitar sobreajuste. Incluso el entrenamiento se hizo en dos etapas: primero se adaptó la capa final encargada de clasificar las lesiones y luego se ajustó todo el modelo de forma más fina. Esto permitió aprovechar el conocimiento previo sin perder precisión.

El resultado final es un sistema capaz de estimar probabilidades para cada tipo de lesión a partir de una imagen. Es importante aclarar que estas probabilidades representan una estimación estadística del modelo, no un diagnóstico médico definitivo. La herramienta busca servir como apoyo para detección temprana y análisis complementario, no reemplazar la evaluación profesional.
